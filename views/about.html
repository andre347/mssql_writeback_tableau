This Write Back to Microsoft SQL Server (v1.0) extension was created to allow you to add data to your SQL Server from within Tableau. This allows you to continue your analysis without having to leave Tableau.

Created by Andre de Vries

How to use:

  1. Click the login tab to login to your Microsoft SQL Server
  2. Go to the data tab to enter the values for the selected columns. You can select a column by checking the corresponding box.
  3. Click the Apply button to write your data back to your database. After your submitted your data you can click 'refresh' to refresh your data source.
